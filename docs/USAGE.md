# 使い方

Search Tabs Customizer の詳しい使い方を説明します．

## 基本的な使い方

### 1. 設定画面を開く

1. Google, Bing, Yahoo! Japan のいずれかで検索を実行
2. 検索結果ページに表示される拡張機能アイコンをクリック
3. モーダル設定画面が開きます

### 2. タブを並び替える

1. 設定画面の「並び替え」ボタンをクリック
2. リスト内のタブをドラッグ&ドロップして順序を変更
3. 設定がリアルタイムで検索結果ページに反映されます

### 3. タブの表示/非表示を切り替える

1. 設定画面の「表示」ボタンをクリック
2. 各タブの右側にある目アイコンをクリックして表示/非表示を切り替え
3. 設定がリアルタイムで検索結果ページに反映されます

### 4. デフォルトに戻す

設定を初期状態に戻したい場合:

1. 設定画面の「デフォルト」ボタンをクリック
2. 確認ダイアログで「OK」をクリック
3. すべての設定が初期状態に戻ります

## 機能詳細

### タブの並び替え機能

対応タブの種類:

- 通常のタブ（すべて, 画像, ニュース, 動画など）
- 「もっと見る」ボタン内のサブタブ

操作方法:

1. 「並び替え」ボタンをクリックして編集モードに入る
2. タブをドラッグして目的の位置にドロップ
3. 編集が完了したら再度「並び替え」ボタンをクリック

注意点:

- 並び替えは各検索サイトごとに個別に保存されます
- アクティブなタブ（現在選択中のタブ）も並び替え可能です
- 設定はリアルタイムで反映されるため，ページのリロードは不要です

### タブの表示/非表示機能

操作方法:

1. 「表示」ボタンをクリックして編集モードに入る
2. 各タブの右側に表示される目アイコンをクリック
3. 設定は自動的に保存されます

注意点:

- 非表示にしたタブは検索結果ページから即座に削除されます
- 設定はリアルタイムで反映されるため，ページのリロードは不要です

### 設定の保存

保存と反映のタイミング:

- タブの並び替え: 「並び替え」ボタンをクリックして編集モードを終了した瞬間にリアルタイムで反映
- 表示/非表示: 目アイコンをクリックした瞬間にリアルタイムで反映
- デフォルトに戻す: 「デフォルト」ボタンをクリックした瞬間にリアルタイムで反映

保存される情報:

- タブの順序
- 各タブの表示/非表示状態

保存場所:

- Chrome のローカルストレージ（`chrome.storage.local`）
- 各検索サイトごとに個別のキーで保存

### 設定モーダル

タブメニュー:

- 設定: タブのカスタマイズを行います
- ドキュメント: 使い方や機能の説明を確認できます
- バージョン: 更新履歴を確認できます
- 情報: 拡張機能の詳細情報や権限を確認できます

出力パネル:
画面下部に表示される出力パネルには，操作ログやエラーメッセージが表示されます．

パネル操作:

- クリア: ログをクリアします
- 最大化/最小化: パネルのサイズを変更します
- 閉じる: モーダル全体を閉じます

## トラブルシューティング

### 拡張機能が動作しない場合

1. 拡張機能の状態を確認:

   - `chrome://extensions/` を開く
   - Search Tabs Customizer が有効になっているか確認
   - エラーメッセージが表示されていないか確認

2. 権限を確認:

   - 拡張機能に必要な権限が許可されているか確認
   - ストレージ権限とタブ権限が必要です

3. ブラウザを再起動:

   - Chrome を完全に終了して再起動

4. 拡張機能を再インストール:
   - 拡張機能をアンインストール
   - 再度インストールして設定を行う

### よくあるエラーと対処法

エラー: "タブ情報を取得できません"

- 原因: ページの読み込みが完了していない，または検索サイトの仕様変更
- 対処: 少し待ってから再度試してください

エラー: "設定の保存に失敗しました"

- 原因: ストレージ権限がない，またはストレージがいっぱい
- 対処: 拡張機能の権限を確認し，ブラウザのストレージ容量を確認してください

タブの順序が元に戻る

- 原因: 設定が正しく保存されていない
- 対処: 「並び替え」ボタンをクリックして編集モードを確実に終了してください
